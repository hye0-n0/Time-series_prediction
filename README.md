# 고등어 가격 예측을 위한 시계열 회귀분석
1. 데이터 설명
- 고등어 가격 예측 모형 추정을 위해 신선 고등어의 소매가격에 대한 2014년 1월부터 2021년 12월 동안의 월별 가격자료를 이용
- 소매가격은 전국 대형마트, 전통시장 등에서 소비자에게 판매하는 평균 가격으로 중품 1마리 당 가격
- 2019년 6월, 2020년 5월, 2020년 6월 평균 가격은 측정되지 않아 시계열 날짜를 기준으로 한 결측치 보간 방법(interpolate metho)을 이용

2. 모형
  1. 다중회귀모형의 자기회귀모형
    - y_t=β_0+β_1 〖 x〗_(1,t)+β_2 〖 x〗_(2,t)+⋯+u_t
    - u_t=ϕ_1 u_(t-1)+ϕ_2 u_(t-2)+⋯+ϵ_t
    - ϵ_t  ~ iid  N(0,σ_ϵ^2 )
  
  2. ARIMA(p, d, q)(P, D, Q)s
    - Y_t=β_0+ϕ_1 Y_(t-1)+ϕ_2 Y_(t-2)+...+ ϕ_p Y_(t-p)  +ϵ_t
    - Y_t=μ+ϵ_t-θ_1 ϵ_(t-1)-θ_2 ϵ_(t-2)-…-θ_p ϵ_(t-q)
    - Y_t=β_0+ϕ_1 Y_(t-1)+ϕ_2 Y_(t-2)+...+ ϕ_p Y_(t-p)  -θ_1 ϵ_(t-1)-θ_2 ϵ_(t-2)-…-θ_q ϵ_(t-q)+ϵ_t
    
  3. LSTM
